# POCKET-PAL
PocketPal is a Java console-based personal finance management system designed for students.  
It helps users track expenses, manage budgets, analyze spending habits, and build healthy financial discipline through badges and reports.

## Features:

- Expense tracking with multiple categories
- Needs vs Wants expense classification
- Budget creation and monitoring
- Badge system to encourage good financial habits
- EMI calculator for loans and purchases
- Monthly spending reports and summaries
- File handling for permanent data storage
- Color-coded console output using ANSI codes

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- File Handling
- ANSI Escape Codes (Console Colors)

## How to Run

1. Open Command Prompt / Terminal
2. Navigate to the project folder:
   cd C:\Users\App\pocketpal3
3. Compile the project:
   javac *.java
4. Run the program:
   java Main
